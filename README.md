# R for Biostatistics

A collection of R scripts demonstrating fundamental statistical concepts used in biostatistics and clinical data analysis.

## Topics Covered

- Set Operations
- Random Sampling
- Binomial Distribution
- Poisson Distribution
- Standard Normal Distribution
- t Distribution
- Central Limit Theorem
- One Sample t-Test
- Independent Student's t-Test
- Welch's t-Test
- Paired t-Test
